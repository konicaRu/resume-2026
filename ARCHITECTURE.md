# ARCHITECTURE

## Назначение

Проект хранит материалы для подготовки и ведения резюме Дмитрия Кочетова под разные
аналитические роли на hh.ru:

- продуктовый аналитик;
- аналитик данных;
- BI-аналитик.

В проекте находятся исходники опыта, вакансии, исследование формата резюме, готовые
версии резюме и правила их редактирования.

## Структура

- `vacansii.md` — архив вакансий и отправленных сопроводительных писем.
- `research_rezum.md` — исследование о формате эффективного резюме на hh.ru.
- `резюме_pr_an.pdf` — текущее резюме под продуктового аналитика.
- `cases_a3.md` — банк кейсов из А3.
- `cases-b2b.md` — банк кейсов из b2bcenter.
- `resume_data_analyst_hh.md` — первая версия резюме под аналитика данных.
- `RULES.md` — накопительные правила проекта.
- `MEMORY.md` — журнал состояния и решений.
- `ARCHITECTURE.md` — карта проекта.
- `README.md` — быстрый вход в проект.
- `project-starter.md` — копия стартового шаблона проекта.

## Команды

- `git status -sb` — проверить состояние проекта.
- `git add -A` — подготовить изменения к коммиту, если весь worktree относится к текущей задаче.
- `git commit -m "..."` — сохранить точку истории.
- `git push -u origin main` — отправить проект в GitHub после настройки remote.
