# resume-2026

Рабочий проект для подготовки резюме Дмитрия Кочетова под разные аналитические роли
на hh.ru.

## Цель

Собрать и вести набор резюме под роли:

- продуктовый аналитик;
- аналитик данных;
- BI-аналитик.

Первый критерий готовности: готовое hh-резюме под аналитика данных, правила проекта,
контекстные файлы и GitHub-репозиторий `konicaRu/resume-2026`.

## Быстрый старт

1. Читать `MEMORY.md`, чтобы понять текущий статус и открытые вопросы.
2. Читать `RULES.md`, чтобы не нарушать принятые правила стиля и фактов.
3. Для правки резюме использовать исходники:
   - `vacansii.md`;
   - `research_rezum.md`;
   - `cases_a3.md`;
   - `cases-b2b.md`;
   - `резюме_pr_an.pdf`.
4. Готовые версии резюме хранить отдельными markdown-файлами.

## Текущие артефакты

- `resume_data_analyst_hh.md` — версия под аналитика данных.

## Принципы

- Не выдумывать опыт, стек и цифры.
- Под каждую роль делать отдельную версию резюме.
- Для hh.ru держать заголовок, навыки и первые строки опыта максимально релевантными роли.
- Финальную шлифовку делать humanizer-подходом: живой русский язык без канцелярита и ИИ-штампов.
- Правки пользователя превращать в правила только после явного подтверждения.
